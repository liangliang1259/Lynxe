## 代码格式化

### Java 代码格式化

#### 1. 使用 Spring Java Format

**配置位置**: `pom.xml`

```xml
<plugin>
    <groupId>io.spring.javaformat</groupId>
    <artifactId>spring-javaformat-maven-plugin</artifactId>
    <version>0.0.39</version>
    <configuration>
        <lineLength>120</lineLength>
    </configuration>
    <executions>
        <execution>
            <id>validate-format</id>
            <phase>validate</phase>
            <goals>
                <goal>validate</goal>
            </goals>
        </execution>
    </executions>
</plugin>
```

#### 2. 使用 Spotless

**配置位置**: `pom.xml`

```xml
<plugin>
    <groupId>com.diffplug.spotless</groupId>
    <artifactId>spotless-maven-plugin</artifactId>
    <version>2.44.5</version>
    <configuration>
        <java>
            <removeUnusedImports/>
        </java>
    </configuration>
</plugin>
```

#### 3. 代码风格要求

**缩进**:
- 使用 4 个空格缩进
- 不使用 Tab

```java
// ✅ 正确
public class Example {
    public void method() {
        if (condition) {
            doSomething();
        }
    }
}

// ❌ 错误
public class Example {
	public void method() {	// 使用 Tab
		if (condition) {
			doSomething();
		}
	}
}
```

**行长度**:
- 最大行长度: 120 字符
- 超过时需要换行

```java
// ✅ 正确
String result = someService.someMethod(param1, param2, param3,
        param4, param5);

// ❌ 错误
String result = someService.someMethod(param1, param2, param3, param4, param5);  // 超过 120 字符
```

**大括号**:
- 左大括号不换行
- 右大括号独占一行

```java
// ✅ 正确
if (condition) {
    doSomething();
}

// ❌ 错误
if (condition)
{
    doSomething();
}
```

**空格**:
- 运算符两侧加空格
- 逗号后加空格
- 关键字后加空格

```java
// ✅ 正确
if (a > b && c < d) {
    int result = a + b - c;
}

// ❌ 错误
if (a>b&&c<d) {  // 运算符两侧无空格
    int result=a+b-c;  // 运算符两侧无空格
}
```

## 代码复杂度控制

### 圈复杂度 (Cyclomatic Complexity)

#### 定义
圈复杂度是衡量代码复杂度的指标，值越大表示代码越复杂。

#### 限制要求

| 代码类型 | 最大圈复杂度 |
|----------|--------------|
| 方法 | 10 |
| 类 | 200 |
| 文件 | 500 |

#### 示例

```java
// ✅ 正确 - 圈复杂度 = 3
public String getExecutionStatus(ExecutionRecord record) {
    if (record == null) {
        return "UNKNOWN";
    }
    if (record.isCompleted()) {
        return "COMPLETED";
    }
    return "RUNNING";
}

// ❌ 错误 - 圈复杂度过高
public String getDetailedStatus(ExecutionRecord record) {
    if (record == null) {
        return "UNKNOWN";
    } else if (record.isPending()) {
        if (record.getRetryCount() > 0) {
            if (record.getLastError() != null) {
                return "PENDING_WITH_ERROR";
            } else {
                return "PENDING";
            }
        } else {
            return "INITIAL";
        }
    } else if (record.isRunning()) {
        if (record.getProgress() > 50) {
            if (record.getProgress() > 80) {
                return "ALMOST_DONE";
            } else {
                return "HALFWAY";
            }
        } else {
            return "JUST_STARTED";
        }
    }
    // ... 更多嵌套
}
```

#### 优化方法

```java
// ✅ 优化后 - 提取方法
public String getDetailedStatus(ExecutionRecord record) {
    if (record == null) {
        return "UNKNOWN";
    }
    if (record.isPending()) {
        return getPendingStatus(record);
    }
    if (record.isRunning()) {
        return getRunningStatus(record);
    }
    return "UNKNOWN";
}

private String getPendingStatus(ExecutionRecord record) {
    if (record.getRetryCount() > 0 && record.getLastError() != null) {
        return "PENDING_WITH_ERROR";
    }
    return record.getRetryCount() > 0 ? "PENDING" : "INITIAL";
}

private String getRunningStatus(ExecutionRecord record) {
    if (record.getProgress() > 80) {
        return "ALMOST_DONE";
    }
    if (record.getProgress() > 50) {
        return "HALFWAY";
    }
    return "JUST_STARTED";
}
```

- 方法参数最多 5 个
- 超过时使用对象封装

#### 示例

void shouldReturnPlanTemplate_whenExists() {
    // Arrange - 准备测试数据

    PlanTemplateEntity entity = new PlanTemplateEntity();

## 代码质量目标

### 指标要求

| 指标 | 目标值 | 说明 |
|------|--------|------|
| 代码覆盖率 | ≥ 60% | 核心业务代码 ≥ 80% |
| 圈复杂度 | ≤ 10 | 单个方法 |
| 方法长度 | ≤ 50 行 | 不包括注释和空行 |
| 类长度 | ≤ 500 行 | 不包括注释和空行 |
| 参数数量 | ≤ 5 个 | 单个方法 |
| 重复率 | ≤ 5% | 代码重复检测 |

### 质量门禁

```bash
# 提交前检查
mvn clean verify
pnpm check:all

# 必须通过
- 编译无错误
- 测试全部通过
- 代码格式检查通过
- 类型检查通过
```
