## 工具系统概述

### 设计理念

```
┌─────────────────────────────────────────────────────────────┐
│                    Agent (决策层)                           │
│  - 分析任务                                                 │
│  - 选择工具                                                 │
│  - 调用工具                                                 │
└────────────────────────────┬────────────────────────────────┘
                             │
┌────────────────────────────▼────────────────────────────────┐
│                  ToolRegistry (工具注册表)                  │
│  - 统一工具管理                                             │
│  - 工具查找                                                 │
│  - 工具验证                                                 │
└────────────────────────────┬────────────────────────────────┘
                             │
         ┌───────────────────┴───────────────────┐
         │                                       │
┌────────▼────────┐                    ┌─────────▼────────┐
│  内置工具         │                    │  MCP 工具         │
│  (Built-in)      │                    │  (External)      │
├─────────────────┤                    ├──────────────────┤
│ • 文件系统       │                    │ • 文件系统服务    │
│ • 数据库         │                    │ • GitHub          │
│ • 浏览器         │                    │ • 数据库服务      │
│ • Office         │                    │ • 自定义服务      │
│ • 并行执行       │                    │                  │
└──────────────────┘                    └──────────────────┘
```

### 核心特性

1. **统一接口**: 所有工具实现相同的 `Tool` 接口
2. **动态注册**: 运行时注册和发现工具
3. **可扩展**: 支持内置工具和 MCP 工具
4. **类型安全**: 强类型参数和返回值
5. **错误处理**: 统一的错误处理机制

### Tool 接口

**位置**: `tool/AbstractBaseTool.java`

```java
public interface Tool {

    /**
     * 获取工具名称
     */
    String getName();

    /**
     * 获取工具描述
     */
    String getDescription();

    /**
     * 获取工具 Schema（参数定义）
     */

    /**
     * 执行工具
     */

}
```

### ToolInput

```java
public class ToolInput {
    private String toolName;
    private Map<String, Object> parameters;

    // 获取参数
    public <T> T getParameter(String key, Class<T> type) {
        Object value = parameters.get(key);
        return type.cast(value);
    }

    public String getRequiredString(String key) {
        String value = getParameter(key, String.class);
        if (value == null || value.isEmpty()) {
            throw new IllegalArgumentException("Required parameter: " + key);
        }
        return value;
    }
}
```

### ToolResult

```java
public class ToolResult {
    private boolean success;
    private Object data;
    private String error;
    private Map<String, Object> metadata;

    public static ToolResult success(Object data) {
        ToolResult result = new ToolResult();
        result.setSuccess(true);
        result.setData(data);
        return result;
    }

    public static ToolResult failure(String error) {
        ToolResult result = new ToolResult();
        result.setSuccess(false);
        result.setError(error);
        return result;
    }

    public String toJson() {
        if (success) {
            return "{\"success\":true,\"data\":" + JsonUtils.toJson(data) + "}";
        } else {
            return "{\"success\":false,\"error\":\"" + error + "\"}";
        }
    }
}
```

## 工具注册

### ToolRegistry

**位置**: `tool/ToolRegistry.java`

```java
@Component
public class ToolRegistry {

    private final Map<String, Tool> tools = new ConcurrentHashMap<>();

    /**
     * 注册工具
     */
    public void registerTool(Tool tool) {
        tools.put(tool.getName(), tool);
    }

    /**
     * 获取工具
     */
    public Tool getTool(String toolName) {
        return tools.get(toolName);
    }

    /**
     * 获取所有工具
     */
    public List<Tool> getAllTools() {
        return new ArrayList<>(tools.values());
    }

    /**
     * 检查工具是否存在
     */
    public boolean hasTool(String toolName) {
        return tools.containsKey(toolName);
    }
}
```

### 自动注册

**Spring 自动扫描**:

```java
@Component
public class ToolAutoRegistrar {

    private final ToolRegistry registry;

    @PostConstruct
    public void registerBuiltinTools() {
        // 扫描所有 Tool 实现类
        List<Tool> tools = ApplicationContext.getBeansOfType(Tool.class);
        tools.forEach(registry::registerTool);
    }
}
```

## MCP 工具集成

### MCP 工具特点

1. **动态加载**: 运行时从 MCP 服务器加载
2. **自动发现**: 通过 MCP 协议自动发现工具
3. **统一接口**: 与内置工具使用相同接口
4. **独立命名**: 工具名称格式 `mcp:{serverName}:{toolName}`

### MCP 工具注册

**位置**: `mcp/service/McpToolRegistry.java`

```java
@Component
public class McpToolRegistry {

    private final ToolRegistry toolRegistry;

    /**
     * 从 MCP 服务器加载工具
     */
    public void loadToolsFromServer(String serverName) {
        McpClient client = mcpService.getClient(serverName);
        List<Tool> tools = client.listTools();

        for (Tool tool : tools) {
            // 包装为 MCP 工具
            McpToolWrapper wrapper = new McpToolWrapper(serverName, tool);

            // 注册到全局工具注册表
            toolRegistry.registerTool(wrapper);
        }
    }
}
```

### MCP 工具调用

```java
// mcp/service/McpService.java
public ToolResult executeMcpTool(String fullToolName, Map<String, Object> arguments) {
    // 1. 解析工具名称
    String[] parts = fullToolName.split(":");
    String serverName = parts[1];
    String toolName = parts[2];

    // 2. 获取 MCP 客户端
    McpClient client = getClient(serverName);

    // 3. 调用工具
    return client.callTool(toolName, arguments);
}
```

### 1. Agent 调用工具

```
┌─────────────────────────────────────────────────────────────┐
│              1. Agent 决策调用工具                          │
│  - 分析用户需求                                             │
│  - 选择合适的工具                                           │
│  - 准备工具参数                                             │
└────────────────────────────┬────────────────────────────────┘
                             │
┌────────────────────────────▼────────────────────────────────┐
│              2. ToolRegistry.getTool()                      │
│  - 查找工具                                                 │
│  - 验证工具存在                                             │
└────────────────────────────┬────────────────────────────────┘
                             │
┌────────────────────────────▼────────────────────────────────┐
│              3. Tool.execute()                              │
│  - 参数验证                                                 │

│  - 返回结果                                                 │
└────────────────────────────┬────────────────────────────────┘
                             │
┌────────────────────────────▼────────────────────────────────┐

                             │
┌────────────────────────────▼────────────────────────────────┐
│              5. 返回结果给 Agent                            │
│  - 将工具结果转换为消息                                     │
│  - 发送给 LLM                                               │
└─────────────────────────────────────────────────────────────┘
```

### 2. 工具执行实现

```java
// tool/AbstractBaseTool.java
public abstract class AbstractBaseTool implements Tool {

    @Override
    public final ToolResult execute(ToolInput input) {
        try {
            // 1. 参数验证
            validateParameters(input);

            // 2. 前置处理
            beforeExecute(input);

            // 3. 执行工具逻辑
            Object result = doExecute(input);

            // 4. 后置处理
            afterExecute(input, result);

            // 5. 返回成功结果
            return ToolResult.success(result);

        } catch (Exception e) {
            logger.error("Tool execution failed: {}", getName(), e);
            return ToolResult.failure(e.getMessage());
        }
    }

    /**
     * 子类实现具体的执行逻辑
     */
    protected abstract Object doExecute(ToolInput input);

    /**
     * 参数验证
     */
    protected void validateParameters(ToolInput input) {
        ToolSchema schema = getSchema();
        for (ParameterDefinition param : schema.getParameters()) {
            if (param.isRequired() && !input.getParameters().containsKey(param.getName())) {
                throw new IllegalArgumentException(
                    "Required parameter missing: " + param.getName()
                );
            }
        }
    }

    /**
     * 前置处理（子类可覆盖）
     */
    protected void beforeExecute(ToolInput input) {
        // 默认空实现
    }

    /**
     * 后置处理（子类可覆盖）
     */
    protected void afterExecute(ToolInput input, Object result) {
        // 默认空实现
    }
}
```

## 工具开发指南

### 创建自定义工具

#### 1. 继承 AbstractBaseTool

```java
@Component
public class MyCustomTool extends AbstractBaseTool {

    @Override
    public String getName() {
        return "my-custom-tool";
    }

    @Override
    public String getDescription() {
        return "My custom tool description";
    }

    @Override
    public ToolSchema getSchema() {
        List<ParameterDefinition> params = List.of(
            ParameterDefinition.builder()
                .name("param1")
                .type("string")
                .description("First parameter")
                .required(true)
                .build()
        );

        return new ToolSchema(getName(), getDescription(), params);
    }

    @Override
    protected Object doExecute(ToolInput input) {
        // 获取参数
        String param1 = input.getRequiredString("param1");

        // 执行逻辑
        String result = processData(param1);

        // 返回结果
        return Map.of("result", result);
    }
}
```

#### 2. 注册工具

```java
@Configuration
public class ToolConfiguration {

    @Bean
    public ToolRegistry toolRegistry() {
        ToolRegistry registry = new ToolRegistry();

        // 注册自定义工具
        registry.registerTool(new MyCustomTool());

        return registry;
    }
}
```

#### 3. 国际化配置

创建 `src/main/resources/i18n/tools/my-custom-tool-zh.yml`:

```yaml
name: 我的自定义工具
description: 这个工具用于...
parameters:
  param1:
    name: 参数1
    description: 第一个参数的说明
```
