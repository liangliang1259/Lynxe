### 代码实现

**位置**: `runtime/service/PlanningCoordinator.java`

```java
@Service
public class PlanningCoordinator {

    private final PlanTemplateService planTemplateService;
    private final PlanningFactory planningFactory;
    private final PlanExecutorFactory executorFactory;
    private final PlanExecutionRecorder recorder;

    public String executeAsync(ExecuteByToolNameRequest request) {
        // 1. 加载计划模板
        PlanTemplateConfigVO template = planTemplateService.getTemplate(
            request.getToolName()
        );

        // 2. 创建执行计划
        AbstractExecutionPlan plan = planningFactory.createPlan(template);
        plan.setReplacementParams(request.getReplacementParams());

        // 3. 同步文件
        if (request.getUploadKey() != null) {
            fileUploadService.syncUploadedFilesToPlan(
                plan.getPlanId(),
                request.getUploadKey()
            );
        }

        // 4. 获取执行器并执行
        AbstractPlanExecutor executor = executorFactory.createExecutor(plan);
        String planId = executor.execute(plan);

        return planId;
    }
}
```

### 2. 同步执行流程

同步执行与异步执行类似，但会等待执行完成后返回结果。

```java
public PlanExecutionResult executeSync(ExecuteByToolNameRequest request) {
    // 1-4 步与异步相同
    AbstractPlanExecutor executor = executorFactory.createExecutor(plan);

    // 5. 等待执行完成
    String planId = executor.execute(plan);

    // 6. 轮询等待结果
    while (true) {
        PlanExecutionRecord record = recorder.getRecord(planId);
        if (record.isCompleted()) {
            return record.toResult();
        }
        Thread.sleep(1000); // 等待 1 秒
    }
}
```

### 3. 执行状态跟踪

**前端轮询实现**

```typescript
// ui-vue3/src/composables/usePlanExecution.ts
export function usePlanExecution() {
  const polling = ref(false)

  // 开始轮询
  const startPolling = async (planId: string) => {
    polling.value = true

    while (polling.value) {
      try {
        const record = await getDetails(planId)

        // 更新状态
        updateExecutionState(record)

        // 检查是否完成
        if (record.completed || record.status === 'failed') {
          polling.value = false
          break
        }

        // 等待 1 秒后继续
        await sleep(1000)
      } catch (error) {
        console.error('Polling error:', error)
        polling.value = false
      }
    }
  }

  // 停止轮询
  const stopPolling = () => {
    polling.value = false
  }

  return {
    startPolling,
    stopPolling
  }
}
```

```java
// planning/service/PlanTemplateService.java
@Transactional
public Integer saveToVersionHistory(PlanTemplateConfigVO template) {
    String planTemplateId = template.getPlanTemplateId();
    String newContent = template.getPlanJson();

    // 检查内容是否相同
    boolean isSame = isContentSameAsLatestVersion(planTemplateId, newContent);
    if (isSame) {
        logger.info("Content is same as latest version, skipping save");
        return getLatestVersionIndex(planTemplateId);
    }

    // 获取下一个版本号
    Integer nextVersionIndex = getNextVersionIndex(planTemplateId);

    // 创建版本实体
    PlanTemplateVersion version = new PlanTemplateVersion();
    version.setPlanTemplateId(planTemplateId);
    version.setVersionIndex(nextVersionIndex);
    version.setPlanJson(newContent);
    version.setCreateTime(LocalDateTime.now());

    // 保存
    PlanTemplateVersion saved = versionRepository.save(version);
    return saved.getVersionIndex();
}
```

### 3. 版本比较

Lynxe 使用智能版本比较，避免因格式差异产生冗余版本。

```java
// planning/service/PlanTemplateService.java
private boolean isContentSameAsLatestVersion(String planTemplateId, String newContent) {
    // 获取最新版本
    PlanTemplateVersion latest = versionRepository
        .findTopByPlanTemplateIdOrderByVersionIndexDesc(planTemplateId)
        .orElse(null);

    if (latest == null) {
        return false;
    }

    // 尝试 JSON 语义比较
    try {
        JsonNode newNode = objectMapper.readTree(newContent);
        JsonNode latestNode = objectMapper.readTree(latest.getPlanJson());
        return newNode.equals(latestNode);
    } catch (JsonProcessingException e) {
        // 如果 JSON 解析失败，回退到字符串比较
        return newContent.equals(latest.getPlanJson());
    }
}
```

```java
// agent/BaseAgent.java
protected List<Message> processToolCalls(List<ToolCall> toolCalls, ExecutionContext context) {
    List<Message> resultMessages = new ArrayList<>();

    for (ToolCall toolCall : toolCalls) {
        try {
            // 1. 查找工具
            Tool tool = toolRegistry.getTool(toolCall.getName());
            if (tool == null) {
                throw new ToolExecutionException(toolCall.getName(), "Tool not found");
            }

            // 2. 解析参数
            ToolInput input = parseToolInput(toolCall.getArguments());

            // 3. 执行工具
            ToolResult result = tool.execute(input);

            // 4. 记录 Think-Act
            recorder.addThinkActRecord(context.getStepId(),
                new ThinkActRecord(toolCall, result));

            // 5. 构建响应消息
            resultMessages.add(new ToolResponseMessage(
                toolCall.getId(),
                result.toJson()
            ));

        } catch (Exception e) {
            logger.error("Tool execution failed", e);
            resultMessages.add(new ToolResponseMessage(
                toolCall.getId(),
                "{\"error\": \"" + e.getMessage() + "\"}"
            ));
        }
    }

    return resultMessages;
}
```

### 3. 并行工具调用

Lynxe 支持并行执行多个工具调用，提高执行效率。

```java
// tool/mapreduce/ParallelExecutionService.java
@Service
public class ParallelExecutionService {

    private final ExecutorService executorService;

    public List<ToolResult> executeParallel(List<ToolCall> toolCalls) {
        List<Callable<ToolResult>> tasks = toolCalls.stream()
            .map(toolCall -> (Callable<ToolResult>) () -> {
                Tool tool = toolRegistry.getTool(toolCall.getName());
                ToolInput input = parseToolInput(toolCall.getArguments());
                return tool.execute(input);
            })
            .collect(Collectors.toList());

        try {
            List<Future<ToolResult>> futures = executorService.invokeAll(tasks);
            return futures.stream()
                .map(future -> {
                    try {
                        return future.get();
                    } catch (Exception e) {
                        return ToolResult.failure(e.getMessage());
                    }
                })
                .collect(Collectors.toList());
        } catch (InterruptedException e) {
            Thread.currentThread().interrupt();
            throw new ParallelExecutionException("Parallel execution interrupted", e);
        }
    }
}
```

### 3. MCP 连接管理

```java
// mcp/service/McpService.java
@Service
public class McpService {

    private final Map<String, McpClient> clientCache = new ConcurrentHashMap<>();

    public List<Tool> getMcpTools(String serverName) {
        // 1. 从缓存获取客户端
        McpClient client = clientCache.get(serverName);
        if (client == null || !client.isConnected()) {
            // 2. 创建新连接
            client = connectToServer(serverName);
            clientCache.put(serverName, client);
        }

        // 3. 获取工具列表
        return client.listTools();
    }

    private McpClient connectToServer(String serverName) {
        // 1. 加载配置
        McpServerConfig config = loadConfig(serverName);

        // 2. 根据连接类型创建客户端
        McpClient client;
        switch (config.getConnectionType()) {
            case PROCESS:
                client = createProcessClient(config);
                break;
            case HTTP:
                client = createHttpClient(config);
                break;
            default:
                throw new McpConfigException("Unknown connection type: " +
                    config.getConnectionType());
        }

        // 3. 初始化
        client.initialize();

        return client;
    }
}
```

```java
// runtime/service/FileUploadService.java
@Service
public class FileUploadService {

    private static final String UPLOAD_DIR = "./uploads";
    private static final String WORKSPACE_DIR = "./workspace";

    public FileUploadResult uploadFiles(MultipartFile[] files) {
        // 1. 生成 uploadKey
        String uploadKey = generateUploadKey();

        // 2. 创建上传目录
        Path uploadPath = Paths.get(UPLOAD_DIR, uploadKey);
        Files.createDirectories(uploadPath);

        // 3. 保存文件
        List<FileInfo> uploadedFiles = new ArrayList<>();
        for (MultipartFile file : files) {
            String originalName = file.getOriginalFilename();
            String storedName = generateUniqueName(originalName);
            Path filePath = uploadPath.resolve(storedName);

            file.transferTo(filePath.toFile());

            uploadedFiles.add(FileInfo.builder()
                .originalName(originalName)
                .storedName(storedName)
                .size(file.getSize())
                .type(file.getContentType())
                .build());
        }

        // 4. 返回结果
        return FileUploadResult.builder()
            .uploadKey(uploadKey)
            .uploadedFiles(uploadedFiles)
            .build();
    }

    public void syncUploadedFilesToPlan(String planId, String uploadKey) {
        // 1. 源目录和目标目录
        Path sourceDir = Paths.get(UPLOAD_DIR, uploadKey);
        Path targetDir = Paths.get(WORKSPACE_DIR, planId, "files");

        // 2. 创建目标目录
        Files.createDirectories(targetDir);

        // 3. 复制文件
        try (Stream<Path> stream = Files.walk(sourceDir)) {
            stream.filter(Files::isRegularFile)
                .forEach(source -> {
                    try {
                        Path target = targetDir.resolve(source.getFileName().toString());
                        Files.copy(source, target, StandardCopyOption.REPLACE_EXISTING);
                    } catch (IOException e) {
                        throw new FileSyncException("Failed to copy file: " + source, e);
                    }
                });
        }
    }
}
```

### 2. 异常恢复机制

```java
// runtime/service/PlanningCoordinator.java
@Service
public class PlanningCoordinator {

    @EventListener
    public void handlePlanException(PlanExceptionEvent event) {
        String planId = event.getPlanId();

        // 1. 记录异常
        logger.error("Plan execution failed: planId={}, error={}",
            planId, event.getMessage());

        // 2. 更新执行状态
        PlanExecutionRecord record = recorder.getRecord(planId);
        if (record != null) {
            record.setStatus(ExecutionStatus.FAILED);
            record.setErrorMessage(event.getMessage());
            recorder.updateRecord(record);
        }

        // 3. 清理资源
        cleanupResources(planId);

        // 4. 发送通知（可选）
        sendFailureNotification(planId, event.getMessage());
    }

    private void cleanupResources(String planId) {
        // 1. 删除临时文件
        Path workspaceDir = Paths.get("./workspace", planId);
        if (Files.exists(workspaceDir)) {
            try {
                FileUtils.deleteDirectory(workspaceDir.toFile());
            } catch (IOException e) {
                logger.warn("Failed to cleanup workspace: {}", planId, e);
            }
        }

        // 2. 释放执行器
        executorPool.releaseExecutor(planId);
    }
}
```
